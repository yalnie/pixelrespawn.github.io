# Introduction

## What is Linkboy?

Linkboy (aka Linking Boy on iOS) is a [GBA](https://en.wikipedia.org/wiki/Game_Boy_Advance) emulator, making it possible to run classic games on modern mobile devices. Among many other things, it emulates [Game Link Cable](https://en.wikipedia.org/wiki/Game_Link_Cable), allowing you to play multi-player games through local networks.
